
omegle-crawler-node
===================

Node library to connect to and interact with the Omegle website.

Getting started
---------------

### Installation

To use omegle-node-crawler in your project, run:

```
npm i omegle-node-crawler
```

or

```
yarn add omegle-node-crawler
```
